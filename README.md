Scientific Calculator in C (Modular Project)

Modules:
- Arithmetic: +, -, *, /
- Scientific: sin, cos, log, power

To Run:
1. Open Code::Blocks
2. Create a new Console Project
3. Add all .c and .h files to the project
4. Link math library (-lm): Project > Build options > Linker settings > Add 'm'
5. Build and run the project
